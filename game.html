CAPSTONE PROJECT

This project is a fast-paced 2D platformer developed in Unity, where players race to complete levels as quickly as possible while maintaining combos and avoiding enemies. The game was built over a 12-week period as part of my final semester capstone project.

The images below showcase the game's user interface, including the splash screen, main menu, level selection screen, pause menu, and various examples of in-game artwork.

At the bottom of the page, you’ll also find a sample UI design created for a separate pixel art game, demonstrating additional design versatility.
